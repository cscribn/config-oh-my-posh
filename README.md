# config-oh-my-posh

## themes/powerlevel10k_amped_lean.omp.json

A Powerlevel10k Lean Oh My Posh theme that uses ANSI colors. Allows your prompt to take on the colors of your terminal profile. Supports dotnet instead of julia.
